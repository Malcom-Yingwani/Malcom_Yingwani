About

A little bit about myself : Becoming a Software Developer

  • As long I Can Remember

    The Desire To Build

    "Tony Stark built this in a cave! With a box of scraps!" - Not Tony Stark

    Just like that my brain has been hooked eversince on the idea of building stuff. Physical or digital, doesn't matter, as long as I could built. For as long as I can remember I've had the urge to build.

  • 2023 - August 2025

    Frustration

    In order to pursue this end, I enrolled to study Mechanical Engineering — the jack of all trades concerning all things building — at the University of the Witwatersrand. For three years I was frustrated; frustrated at a theoratical course that only sought to get more theoratical as it progressed. Three years and I had built nothing. So in August of 2025, I left.

  • September 2025

    Embracing Software Development

    Theory about a hammer is good; as far as it equips you to hammer.

    What now? Software Development. A space where I would be allowed to build, a space I now realised I would love. I enrolled for the September 2025 cohort of the "Java Software Engineering" bootcamp programme offered by Code College ZA. Their practical, project based philosphy of learning was the selling piont.

ReadUP


ReadUP is a full-stack web application built with Python and Django that allows users to add and manage books through authenticated accounts. The project focuses on practical backend development, including clean data models, CRUD operations, and Django’s built-in authentication system. It is deployed as a production-ready application with a simple, responsive interface.


Tech Stack

  • Python, Django
  • HTML5, HTMX
  • CSS3, Bootstrap
  • JavaScript
  • SQLite, PostgreSQL (Production database)
  • Render (Deployment)

ContactHub


ContactHub is a backend-focused web application built with Python and Django for managing personal or business contacts. While the application is currently offline, it was previously deployed on Render and supported media uploads via Cloudinary, demonstrating experience with third-party integrations and production configuration. The project focuses on core CRUD functionality, relational data modeling, and clean backend structure, similar to real-world internal tools and admin systems.


Tech Stack

  • Python, Django
  • HTML5, HTMX
  • CSS3, Tailwind CSS
  • JavaScript
  • Cloudinary (Media Uploads – previously)
  • Render (Deployment – previously)
